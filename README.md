# Prerana_Restaurant_Recommendation_System_ML
Developed a Machine Learning–based Restaurant Recommendation System that suggests restaurants based on user preferences such as cuisine, ratings, cost, and location. Built and trained models to analyze patterns and integrated them with a Flask web application for real-time, user-friendly recommendations.

Problem Statement
In today’s fast-paced lifestyle, users often struggle to choose the right restaurant due to the availability of numerous options. There is a need for a smart system that can filter and recommend restaurants based on individual preferences, saving time and improving decision-making.

Objectives
• To create a smart recommendation engine for restaurant selection
• To process and analyze restaurant-related data efficiently
• To build a predictive model using machine learning techniques
• To design a simple and interactive web interface
• To enhance user experience by providing quick suggestions

Key Features
• Intelligent filtering based on user inputs
• Fast and accurate recommendation system
• Interactive and responsive UI
• Supports multiple preference inputs
• Seamless integration of ML model with web app

Technologies Used
• Python
• Flask Framework
• HTML, CSS
• Scikit-learn
• Pandas, NumPy
• Pickle
